TypForge

Create, edit, and render cover letters using AI-enhanced Typst.

TypForge is a tool for rapidly prototyping job-specific typeset cover letter templates.
It's not intended to write an entire AI slop letter for you, rather it helps targeting specific talking points using the application listing and your previous experience (CV).
This is why I've integrated a Typst editing environment with live incremental compilation into TypForge, so you can enter the actual content.

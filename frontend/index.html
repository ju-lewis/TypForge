TypForge

Create, edit, and render cover letters using AI-enhanced Typst.

TypForge is a tool for rapidly prototyping application-specific typeset cover letter templates.
It's not intended to write an entire AI slop letter for you, rather it helps targeting specific talking points using the application listing and your previous experience (CV).
This is why I've integrated a very minimal Typst editing environment with live incremental compilation into TypForge, so you can write the content direcly in the app.
You can also directly export the .typ file for editing in a more capable editor.
